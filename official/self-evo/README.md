## Corpus bridge set

- Explicit bridge: `c = a + b`.
- Information theory bridge: package-control files preserve hashes, versions, caveats, and nonclaims so the package does not compress away safety-critical uncertainty.
- Cybernetics bridge: self-evolution requires visible control topology; SRLM, CGAM, TRIAD, Memory Gate, Local Checker, fixture runner, contradiction register, and human gate must not collapse into one controller.
- Physiology bridge: review findings and append-first corrections are scar tissue, not decoration; they preserve memory of weak points without authorizing growth.
- Earth paragraph: this package is like a drawing register and commissioning checklist. It organizes revisions and gates. It does not energize the system.

## Boundary

These materials are document-level governance artifacts.

They are not:

- a conformance certificate;
- deployment authorization;
- a safety proof;
- a personhood or consciousness claim;
- live self-evolution permission;
- autonomous resource-acquisition permission.
