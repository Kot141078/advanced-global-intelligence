The **c Hardening Pack v0.1** is a cross-corpus hardening package for the `c = a + b` architecture.

## What this package does

This package:

* defines hardening profiles for post-anchor continuity, anti-autarky, claim discipline, local infrastructure, public experiments, physical endpoints, memorial interfaces, and resource actors;
* provides machine-facing and human-facing package control;
* preserves the distinction between architecture, implementation, evidence, authority, continuity, personhood, and economic claims;
* prevents new strong concepts from becoming sources of drift or overclaiming.

## What this package does not claim

This package does **not** claim:

* legal personhood for `c`;
* proof of consciousness;
* proof of AGI in the legacy Artificial General Intelligence sense;
* proof that a system is safe in all environments;
* automatic post-anchor authority;
* automatic re-anchoring;
* self-funded autonomous growth;
* hardware sovereignty;
* public experiment proof of personhood or model capability;
* replacement for law, institutions, medicine, courts, or human responsibility.
