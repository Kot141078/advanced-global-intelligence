## Purpose

This directory contains the AMDR/PAMDC addendum to the Child-`c` Development Protocol corpus.

It adds a bounded memory and continuity layer focused on:

- active memory degradation while `a` remains present;
- memory freshness and recalibration;
- lifestream ingestion discipline;
- Judge integration as evidence rather than authority;
- write-path failure detection;
- post-anchor memory degradation;
- post-anchor continuity, archive, replay, fork, and resume discipline.

This is an addendum-only delta.

It does **not** replace, rewrite, or supersede:

- CCDP v0.1 Full Technical Corpus;
- CCDP v0.1.1 Hygiene Addendum;
- baseline CCDP schemas;
- baseline CCDP technical/sensitive profiles.
