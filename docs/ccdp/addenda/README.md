# CCDP Addenda

This directory contains post-baseline addenda to the Child-`c` Development Protocol corpus.

Addenda are bounded extensions or corrective packages. They do not automatically replace the CCDP v0.1 baseline or the CCDP v0.1.1 Hygiene Addendum.
