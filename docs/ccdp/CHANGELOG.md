### Added

- `CCDP_v0_1_1_Hygiene_Patch.md`.

The patch does not add a new safety feature. It defines release hygiene corrections:

- obsolete draft isolation;
- Traceability Matrix refresh;
- Conformance Matrix refresh;
- canonical precedence rule;
- clean-start clarification;
- terminology disambiguation;
- raw-evidence exception centralization;
- Soft Safety Red/Black `MUST` clarification;
- jurisdictional handoff reminders;
- public / technical / sensitive split;
- release notes and integrity checklist.
