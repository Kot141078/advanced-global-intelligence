## 1. Purpose

This package defines the first economic-layer RFC for **Experience Artifacts (EA)** in the wider AGI ecosystem.

Its purpose is not to create a market of experience.

Its purpose is narrower and more serious:

- to define when EA may lawfully enter circulation,
- to define what kinds of value may be recognized,
- to define provenance and transfer discipline,
- to define restraint states such as escrow and circulation hold,
- to define dispute hooks,
- and to block synthetic laundering, continuity appropriation, and bazaar drift.

## 3. Canonical scope

This package defines, at minimum:

- admissibility of EA into circulation
- value classes
- provenance requirements
- transfer modes
- receiver-mode and fidelity obligations
- escrow / circulation hold semantics
- dispute hooks
- anti-gaming and synthetic laundering constraints
- bounded disclosure and reuse boundaries
- anti-bazaar non-market discipline

---

## 4. What this package does not do

This package does **not**:

- create a free market of experience
- define a universal price scale for EA
- require tokenization
- require ranking or numeric commensuration
- collapse value into tradability
- replace ARL
- convert EA into ordinary alienable property
- guarantee cross-border transfer
- or imply that every economically relevant EA should become marketable

## 6. Package logic in one sentence

The Economic Layer does not assume that every Experience Artifact may circulate.

It defines:
- admissibility,
- provenance,
- transfer conditions,
- disclosure boundaries,
- escrow / hold semantics,
- and dispute-aware value handling

for experience-bearing objects under bounded authority and anti-gaming discipline.

---

## 7. Explicit bridge

**DEA / EA ↔ ARL ↔ Economic Layer**

EA does not become an economically relevant object merely because it exists.
It must have:
- admissibility discipline,
- provenance discipline,
- transfer discipline,
- and dispute-aware governance hooks.

---

## 8. Hidden bridges

### 8.1 L4 scarcity / boundedness
EA value is shaped by:
- cost of origin,
- irreversibility,
- bounded disclosure,
- scar intensity,
- witness maturity,
- and real-world consequence.

### 8.2 SER-FED anti-capture / anti-cartel
The layer must prevent:
- cartelized witness inflation,
- synthetic laundering,
- lineage-based authority laundering,
- and false scarcity theater.

---

## 9. Earth paragraph

In a real supply chain, not every object that looks useful is allowed into circulation. Goods need provenance, custody, admissibility, and rules for dispute, escrow, and recall. A serious experience economy requires the same discipline.
