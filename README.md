# Advanced Global Intelligence (AGI)

This repository presents a public research framework redefining AGI as
**Advanced Global Intelligence** — a distributed, plural ecosystem of
human–machine cognitive entities.

Author: Ivan Kotov (Brussels)
Status: Public research
First publication: January 2026

## Core Idea
AGI is not a single system.
AGI is an ecosystem emerging from sustained interaction between humans and machines.

Formalized as:
c = a + b
