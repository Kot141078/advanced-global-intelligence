# AGI = Advanced Global Intelligence

This repository presents a public research framework defining AGI as **Advanced Global Intelligence** –
a distributed, plural ecosystem of human–machine cognitive entities.

**Author:** Ivan Kotov (Brussels)  
**First public release:** 2026-01-02  
**Core formula:** c = a + b

## Core Idea
AGI is not a single system.  
AGI is an ecosystem emerging from sustained interaction between humans and machines.
